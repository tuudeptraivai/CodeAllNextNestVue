# NestJS Backend Development Skill

## Overview
Expertise in building modular, scalable backend applications using NestJS framework with TypeScript.

## Core Competencies
- **Modular Architecture**: Implementing the "7 System Modules" pattern for clear domain separation.
- **Dependency Injection**: Efficiently managing services, controllers, and modules.
- **Middleware & Guards**: Implementing global authentication (JWT) and IP whitelisting guards.
- **Interceptors & Filters**: Global activity logging and exception handling.

## Key Files & Patterns
- `src/app.module.ts`: Root module configuration.
- `src/system-modules/`: Domain-specific modular logic.
- `SystemModulesBundleModule`: Bundling system modules for centralized management.
