# Kỹ Năng: Automated Crawl & Data Ingestion

## Mô Tả
Xây dựng công cụ thu thập dữ liệu tự động từ các nguồn bên ngoài để làm giàu nội dung cho hệ thống phim.

## Các Công Nghệ Chính
- **Axios:** Thực hiện các yêu cầu lấy dữ liệu từ API bên ngoài.
- **React Query:** Quản lý trạng thái và đồng bộ dữ liệu crawl.

## Khả Năng
- Lập lịch và thực hiện crawl phim từ nhiều nguồn khác nhau.
- Xử lý và chuẩn hóa dữ liệu phim (tên, mô tả, ảnh, link) trước khi lưu vào hệ thống.
- Giao diện điều khiển crawl trực quan, cho phép admin theo dõi tiến độ và xử lý lỗi.
- Tích hợp công cụ crawl external cho các trường hợp đặc biệt.
