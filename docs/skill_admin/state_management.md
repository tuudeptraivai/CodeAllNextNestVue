# Kỹ Năng: State Management & Data Fetching

## Mô Tả
Quản lý trạng thái ứng dụng lớn và tối ưu hóa việc lấy dữ liệu từ server.

## Các Công Nghệ Chính
- **Redux Toolkit:** Quản lý global state (auth, settings, system status).
- **React Query (v3):** Quản lý server state, tự động cache và refetch dữ liệu.

## Khả Năng
- Đồng bộ hóa trạng thái giữa các component khác nhau trong hệ thống.
- Tối ưu hóa hiệu suất bằng cách giảm thiểu số lần gọi API thừa thông qua caching.
- Xử lý các tác vụ bất đồng bộ (Async Thunks) một cách khoa học.
- Quản lý session đăng nhập và phân quyền người dùng (Role-based Access Control).
