# Kỹ Năng: Data Visualization & System Tools

## Mô Tả
Trực quan hóa dữ liệu hệ thống và cung cấp các công cụ vận hành chuyên sâu cho Admin.

## Các Công Nghệ Chính
- **Recharts:** Vẽ biểu đồ thống kê truy cập, số lượng phim, v.v.
- **XLSX:** Xuất báo cáo dữ liệu ra file Excel.
- **Redis Management:** Giao diện quản lý bộ nhớ đệm.

## Khả Năng
- Tạo Dashboard theo dõi sức khỏe hệ thống theo thời gian thực.
- Xuất danh sách phim, người dùng, báo cáo tài chính/thống kê ra file Excel để lưu trữ và phân tích.
- Quản lý và làm sạch cache Redis trực tiếp từ giao diện CMS.
- Theo dõi và điều phối các tác vụ chạy ngầm (Background Jobs).
