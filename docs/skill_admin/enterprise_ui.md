# Kỹ Năng: Enterprise UI Development (Ant Design Pro)

## Mô Tả
Phát triển giao diện quản trị phức tạp, có tính nhất quán cao và hiệu suất tốt cho môi trường doanh nghiệp.

## Các Công Nghệ Chính
- **Ant Design (antd):** Thư viện UI component chuẩn mực.
- **Ant Design Pro Components:** Các component cấp cao như ProTable, ProForm giúp xây dựng trang quản trị cực nhanh.

## Khả Năng
- Xây dựng các bảng dữ liệu (ProTable) với đầy đủ tính năng lọc, tìm kiếm, phân trang và xuất dữ liệu.
- Thiết kế layout quản trị chuyên nghiệp với Sidebar, Header và Breadcrumbs tự động.
- Xử lý các form phức tạp với validation chặt chẽ.
- Tạo các modal và thông báo (toast) tương tác với người dùng mượt mà.
