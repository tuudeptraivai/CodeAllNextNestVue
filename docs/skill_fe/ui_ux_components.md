# Kỹ Năng: UI/UX & Responsive Design

## Mô Tả
Xây dựng giao diện người dùng hiện đại, đẹp mắt và hoàn toàn tương thích với nhiều thiết bị (Desktop, Tablet, Mobile).

## Các Công Nghệ Chính
- **Tailwind CSS:** Framework CSS utility-first để thiết kế nhanh và linh hoạt.
- **Swiper & Keen-slider:** Tạo các hiệu ứng chuyển động và slider phim mượt mà.
- **Lucide & React Icons:** Bộ icon vector sắc nét.

## Khả Năng
- Giao diện Responsive hoàn toàn.
- Hiệu ứng Hover và chuyển cảnh mượt mà.
- Tối ưu hóa trải nghiệm người dùng trên thiết bị di động.
- Sử dụng Dark mode hoặc các bảng màu tùy biến (nếu có).
