# Kỹ Năng: Data Fetching & State Management

## Mô Tả
Xử lý việc lấy dữ liệu từ API một cách hiệu quả, hỗ trợ caching và đồng bộ hóa trạng thái ứng dụng mượt mà.

## Các Công Nghệ Chính
- **TanStack Query (React Query):** Quản lý server state, caching dữ liệu.
- **SWR:** Lấy dữ liệu nhanh và cập nhật tự động.
- **Axios:** Thư viện HTTP client để gửi yêu cầu đến server.

## Khả Năng
- Caching dữ liệu để giảm tải cho server và tăng tốc ứng dụng.
- Tự động lấy lại dữ liệu khi kết nối mạng được khôi phục hoặc cửa sổ trình duyệt được focus.
- Xử lý trạng thái Loading và Error một cách chuyên nghiệp.
- Phân trang dữ liệu hiệu quả cho danh sách phim.
